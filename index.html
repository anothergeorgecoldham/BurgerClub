How it works

Stage 1 collects opt-ins. Stage 2 chooses the most popular day and meal time, then schedules one catchup per month using each person’s burger joint suggestion in rotation.

Address lookup uses the free OpenStreetMap Nominatim search. You can also type an address manually if lookup is unavailable.
